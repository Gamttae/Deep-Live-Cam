## 1. 기기 소개

**Item Finder**는 자주 잃어버리는 물건(리모컨, 안경, 지갑, 약통 등)에  
작은 태그를 붙여두고, 잃어버렸을 때 **방향과 거리**를 동시에 알려주는 탁상형 기기입니다.

### 기기 구성 요소

```
┌─────────────────────────────────────────┐
│                                         │
│  ┌──────────────────────────────────┐   │
│  │  LCD 화면 (20글자 × 4줄)         │   │
│  │  - 메뉴, 방향, 거리 표시         │   │
│  └──────────────────────────────────┘   │
│                                         │
│  ┌──────┐   ┌───────────────────────┐   │
│  │키패드│   │  방향 지시 모터       │   │
│  │(4×4) │   │  (Flipper Zero 얹힘)  │   │
│  └──────┘   └───────────────────────┘   │
│                                         │
│  🔊 부저 (거리 비프음)                  │
└─────────────────────────────────────────┘
```

### 물건에 붙이는 태그

```
┌─────────────────┐
│  NFC 스티커     │  ← 얇고 작음, 물건 표면에 부착
│  +              │
│  BLE 비콘       │  ← 버튼 전지형, 신호 지속 발신
└─────────────────┘
```

## 2. 처음 시작하기

### 전원 켜기 순서

1. **Flipper Zero** 전원을 켜고 `앱 → Tools → Item Finder` 를 실행합니다.
   - Flipper Zero 화면에 "Item Finder — 대기 중" 표시 확인
2. **기기 본체** USB 전원을 연결합니다.
   - LCD에 아래 화면이 나타나면 준비 완료입니다.

```
┌────────────────────┐
│=== Item Finder ===│
│1: 물건 등록        │
│2: 물건 찾기        │
│등록된 물건: 0개    │
└────────────────────┘
```

> Flipper Zero를 먼저 실행하지 않으면 NFC/BLE 기능이 동작하지 않습니다.

## 3. 물건 등록하기

물건을 처음 등록할 때 한 번만 하면 됩니다.  
물건에 **NFC 태그**와 **BLE 비콘**이 모두 부착되어 있어야 합니다.

### 단계별 안내

#### 단계 1 — NFC 태그 읽기

1. 키패드에서 **`1`** 을 누릅니다.
2. LCD에 아래 메시지가 표시됩니다:

```
┌────────────────────┐
│단계 1/3: NFC 등록 │
│물건에 붙인 NFC 태그│
│Flipper에 태그하세요│
│                    │
└────────────────────┘
```

3. 등록할 물건을 들고 **Flipper Zero 뒷면 하단**(NFC 안테나 위치)에  
   NFC 태그를 **3cm 이내**로 가져다 댑니다.
4. Flipper Zero가 진동하면 인식 성공입니다.

#### 단계 2 — 물건 이름 입력

1. LCD에 NFC UID가 표시되고 이름 입력 화면이 나옵니다:

```
┌────────────────────┐
│단계 2/3: 이름 입력│
│04:A3:2B:C1        │
│이름 입력 후 # 확인│
│_                  │
└────────────────────┘
```

2. 키패드 숫자로 이름을 입력합니다. (예: 리모컨 = `7`, `4`, `6`, `6`)
3. 입력 중 실수하면 **`*`** 로 한 글자씩 지웁니다.
4. 입력 완료 후 **`#`** 을 누릅니다.

> **이름 입력 팁**: 숫자키로만 입력됩니다.  
> 물건별로 간단한 숫자 코드를 정해두세요 (예: 1=리모컨, 2=안경, 3=지갑).

#### 단계 3 — BLE 비콘 등록

1. LCD에 아래 메시지가 표시됩니다:

```
┌────────────────────┐
│단계 3/3: BLE 등록 │
│BLE 태그를 Flipper  │
│가까이 가져오세요   │
│                    │
└────────────────────┘
```

2. 물건에 붙인 **BLE 비콘**을 Flipper Zero에 **50cm 이내**로 가져갑니다.
3. Flipper Zero가 BLE 신호를 감지하면 자동으로 등록됩니다.
4. LCD에 "등록 완료!" 메시지와 함께 물건 이름, MAC 주소가 표시됩니다.

```
┌────────────────────┐
│등록 완료!          │
│리모컨              │
│AA:BB:CC:DD:EE:FF  │
│아무 키 → 메뉴     │
└────────────────────┘
```

> 최대 **8개**의 물건을 등록할 수 있습니다.

## 4. 물건 찾기

### 찾기 시작

1. 키패드에서 **`2`** 를 누릅니다.
2. 등록된 물건 목록이 나타납니다:

```
┌────────────────────┐
│2/8=이동  #=선택  *=뒤│
│>리모컨             │
│ 안경               │
│ 지갑               │
└────────────────────┘
```

3. **`2`** 로 아래로 이동, **`8`** 로 위로 이동하여 찾을 물건을 선택합니다.
4. **`#`** 을 눌러 스캔을 시작합니다.

### 360° 스캔 중

기기가 자동으로 Flipper Zero를 360°로 돌리며 BLE 신호를 측정합니다.  
약 15~20초가 걸립니다. 기다려 주세요.

```
┌────────────────────┐
│스캔 중...          │
│물건: 리모컨        │
│각도: 090°          │
│RSSI: -72 dBm      │
└────────────────────┘
```

### 스캔 완료 — 방향·거리 표시

스캔이 끝나면 모터가 **물건 방향**을 가리키고 LCD에 결과가 표시됩니다:

```
┌────────────────────┐
│리모컨              │
│방향: 135°          │
│거리: ~3m  -72dBm   │
│* 종료              │
└────────────────────┘
```

- **방향(°)**: 모터가 가리키는 각도. 기기 정면을 0°로 기준
- **거리(m)**: BLE RSSI로 추정한 대략적 거리 (±50% 오차 있음)
- **부저**: 물건 쪽으로 이동할수록 비프 소리가 빨라짐

### 물건 근처 탐색

부저 소리를 들으며 LCD가 가리키는 방향으로 이동합니다.

| 비프 속도 | 의미 |
|---|---|
| 느림 (2초마다 한 번) | 멀리 있음 (5m 이상) |
| 보통 (1초마다 한 번) | 중간 거리 (2~5m) |
| 빠름 (0.5초마다) | 가까이 있음 (1~2m) |
| 매우 빠름 (0.2초마다) | 바로 근처 (50cm 이내) |

### 찾기 종료

**`*`** 를 누르면 스캔이 중지되고 메인 메뉴로 돌아갑니다.

## 5. LCD 화면 설명

### 메인 메뉴

```
┌────────────────────┐  행 0: 타이틀
│=== Item Finder ===│
│1: 물건 등록        │  행 1: 기능 1
│2: 물건 찾기        │  행 2: 기능 2
│등록된 물건: 3개    │  행 3: 현재 등록 수
└────────────────────┘
```

### 스캔 중 화면

```
┌────────────────────┐
│스캔 중...          │  행 0: 상태
│물건: 지갑          │  행 1: 찾는 물건
│각도: 180°          │  행 2: 현재 스캔 각도
│RSSI: -68 dBm      │  행 3: 현재 신호 강도
└────────────────────┘
```

### 결과 화면

```
┌────────────────────┐
│지갑                │  행 0: 물건 이름
│방향: 270°          │  행 1: 방향 (0~360°)
│거리: ~2m  -65dBm   │  행 2: 거리, 신호 강도
│* 종료              │  행 3: 종료 방법
└────────────────────┘
```

## 6. 키패드 버튼 안내

```
┌────────────────────────┐
│  1    2    3    A      │
│(등록)(↓)  (3)  (-)     │
│                        │
│  4    5    6    B      │
│  (4)  (5)  (6)  (-)    │
│                        │
│  7    8    9    C      │
│  (7)  (↑)  (9)  (-)    │
│                        │
│  *    0    #    D      │
│(취소)(0)(확인)(-)      │
└────────────────────────┘
```

| 버튼 | 기능 |
|---|---|
| `1` | 메인 메뉴: 물건 등록 |
| `2` | 메인 메뉴: 물건 찾기 / 목록에서 아래 이동 |
| `8` | 목록에서 위 이동 |
| `#` | 확인 / 선택 |
| `*` | 취소 / 뒤로 가기 / 찾기 중지 |
| `0`~`9` | 이름 입력 시 숫자 입력 |

> 키패드에 **큰 캡(cap)**을 씌우면 손이 불편한 어르신도 쉽게 누를 수 있습니다.

## 7. 부저 신호 의미

| 상황 | 부저 패턴 |
|---|---|
| 스캔 완료, 물건 멀리 있음 | 낮고 느린 비프 |
| 물건 방향으로 이동 중 | 비프 점점 빨라짐 |
| 물건 바로 근처 (50cm) | 매우 빠른 연속 비프 |
| 오류 발생 | 없음 (LCD에 오류 메시지) |

## 8. 서보 모터(방향 지시기) 안내

- 서보 모터 위에 Flipper Zero가 얹혀 있습니다.
- 스캔 시 Flipper Zero가 15°씩 24회 회전하여 360°를 모두 측정합니다.
- 스캔 완료 후 **가장 강한 BLE 신호 방향**에 멈춥니다.
- 이 방향이 물건이 있는 방향입니다.

> 모터와 물건 방향이 맞지 않는 경우:  
> 서보 속도 보정값(`MS_PER_DEG`)을 코드에서 조정하세요. (기본값: 6ms/°)

## 9. BLE 태그(비콘) 준비

물건에 붙일 BLE 비콘이 필요합니다.  
아래 중 하나를 선택하세요:

### 옵션 A — 상용 BLE 비콘 (가장 쉬움)

- 인터넷에서 "BLE 비콘 버튼 전지" 검색
- iBeacon 또는 Eddystone 프로토콜 지원 제품 구매
- 배터리 교체형, 부착 즉시 사용 가능

### 옵션 B — ESP32-C3 미니 (저렴, DIY)

아래 코드를 ESP32-C3에 업로드하면 BLE 비콘으로 동작합니다:

```cpp
// ESP32-C3 BLE iBeacon 예시 (Arduino IDE)
#include <BLEDevice.h>
#include <BLEAdvertising.h>

void setup() {
    BLEDevice::init("ItemTag");
    BLEAdvertising* adv = BLEDevice::getAdvertising();
    adv->setScanResponse(false);
    adv->setMinPreferred(0x06);
    BLEDevice::startAdvertising();
}

void loop() {
    delay(1000);
}
```

- 크기: 엄지손톱 정도, 배터리: LiPo 100mAh (1~2주 지속)
- 3M 양면 테이프로 물건에 부착

### 옵션 C — 기존 기기 활용

스마트폰을 BLE 비콘 앱으로 전환하여 테스트할 수 있습니다.  
(실 사용 시에는 전용 비콘 권장)

## 10. NFC 태그 준비

- **NTAG213** 또는 **NTAG215** 스티커 태그 사용 (ISO14443-3A)
- 인터넷에서 "NFC 스티커 NTAG213" 검색, 10개 묶음으로 저렴하게 구매 가능
- BLE 비콘 위에 함께 붙이거나, 물건 표면에 별도로 붙임
- 등록 후에는 NFC 태그를 제거해도 찾기 기능(BLE)은 계속 작동

---

## 11. 주의사항

- Flipper Zero를 반드시 먼저 실행한 후 본체 전원을 켜세요.
- BLE 비콘 배터리가 방전되면 찾기 기능이 동작하지 않습니다. 주기적으로 확인하세요.
- BLE 거리 추정은 **대략적인 수치**입니다. 벽·금속·가구로 인해 오차가 발생할 수 있습니다.
- 서보 모터에 손이나 물건이 끼이지 않도록 주의하세요.
- 기기를 습한 곳이나 직사광선 아래 두지 마세요.
- 등록 데이터는 Arduino 메모리에 저장되며, **전원을 끄면 초기화**됩니다.  
  (영구 저장이 필요하면 EEPROM 저장 기능 추가 필요)
